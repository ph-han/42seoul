# miniRT
> 이 과제는 Ray Tracing을 구현해보는 과제이다. <br>
> map파일에 입체적으로 렌더링하고 싶은 물체들, 카메라의 위치/방향, 빛의 위치/세기가 주워진다. <br>
> map에서 주어진 정보들을 이용해 Ray Tracing을 이용해 렌더링한다.<br>
